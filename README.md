# FINN – Web aplikacija za praćenje financija

## 1. NAZIV PROJEKTA

Web aplikacija za praćenje financija.

## 2. PROŠIRENI OPIS WEB APLIKACIJE

Web aplikacija koja omogućuje korisnicima praćenje i upravljanje troškovima. Korisniku je omogućeno unositi financijske podatke poput troškova, kategorije troškova, datuma i opisa.

Pružen je pregled troškova kroz kategorije i vremensko razdoblje. Aplikacija omogućuje filtriranje i pregled troškova prema kategoriji, iznosu i vremenskom intervalu.

Cilj aplikacije je omogućiti jednostavno praćenje financijskih podataka.

## 3. FUNKCIONALNOST WEB APLIKACIJE

### 1. FUNKCIONALNOST

Registracija i autentifikacija korisnika

* Registracija korisnika
* Prijava korisnika
* Odabir tipa korisničkog računa (osobni/poslovni)

### 2. FUNKCIONALNOST

Upravljanje troškovima

* Unos troškova (iznos, kategorija, datum, opis)
* Uređivanje troškova
* Brisanje troškova
* Pregled troškova

### 3. FUNKCIONALNOST

Filtriranje i analiza troškova

* Filtriranje po iznosu, kategoriji i datumu
* Pregled kroz vremenske intervale

### 4. FUNKCIONALNOST

Pregled statističkih podataka

* Statistika potrošnje po kategorijama i razdoblju
* Grafički prikaz potrošnje (grafovi i dijagrami)

### 5. FUNKCIONALNOST

Upravljanje ostalim financijama

* Postavljanje budžeta (mjesečno i godišnje)
* Unos prihoda (predviđenih i nepredviđenih)
